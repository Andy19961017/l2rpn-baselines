# L2RPN_Baselines
Repository hosting reference baselines for the [L2RPN challenge](https://l2rpn.chalearn.org/)

# Install 
## Requirements
`python3 >= 3.6`

## Instal from PyPI
```sh
pip3 install l2rpn_baselines
```
## Install from source
```sh
git clone https://github.com/rte-france/l2rpn-baselines.git
cd l2rpn-baselines
pip3 install -U .
cd ..
rm -rf l2rpn-baselines
```
